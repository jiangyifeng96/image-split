# 图片九宫格分割工具

一个现代化的网页应用，可以将图片分割成九宫格布局，非常适合用于创建社交媒体内容。

## 功能特性

- **图片上传**
  - 支持拖拽上传
  - 支持传统文件选择
  - 支持常见图片格式

- **图片处理**
  - 将图片分割成9个相等的部分
  - 保持原始图片比例
  - 分割结果预览
  - 支持单独下载每个分割后的图片

- **用户体验**
  - 支持PC端和移动端响应式设计
  - 采用苹果风格配色方案
  - 直观的界面设计
  - 实时预览效果

## 技术栈

- 现代JavaScript
